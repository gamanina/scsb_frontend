機器設備升級貸款
立即申辦
產品說明

「機器設備升級貸款」為配合政府產業自動化、環境保護及節約能源政策，鼓勵企業購置自動化、改善汙染防治及節約能源設備，有效防治汙染與促進能源有效利用，以促進產業升級之專案貸款。


申貸資訊
  • 貸款對象：國內之公民營企業。
    • 購置自動化機器設備之投資計畫。
    • 購置電腦軟硬體之投資計畫。
    • 購置汙染防治設備之投資計畫。
    • 購置節約能源設備之投資計畫。
    • 購置利用新及淨潔能源設備之投資計畫。
    • 公路(市區)汽車客運業者更新(新購或汰舊換新)車輛與其相關車內設施計畫。
    • 購置經經濟部認定屬於自動化、節約能源、新及淨潔能源等設備之投資計畫。
  • 貸款期限：最長10年(含寬限期)，寬限期最長3年。
  • 貸款額度：
    • 每一計畫貸款額度，視申請人財務狀況及申貸額度核定，核貸額度最高不得超過該計畫成本之80%。
    • 申請人申請核准適用本貸款額度最高為新臺幣4億元，購置汙染防治設備之投資計畫最高可為新臺幣10億元；每一貸款計畫限同一家承貸銀行承貸，但聯貸案之貸款計畫得由各參貸行共同承貸。
      • 例(1)：A客戶原申貸購置自動化機器設備核准額度4億元，該客戶再申請購置汙染防治設備核准額度最高可達6億元(10億元-4億元)。
      • 例(2)：B客戶僅申貸購置汙染防治設備核准額度最高可達10億元(無購置其他設備計畫)。
      • 例(3)：申貸購置汙染防治設備核准額度1億元時，該客戶再申請購置其他設備計畫(非汙染防治設備)剩餘可用額度最高可達3億元(4億元-1億元)。
    • 貸款利率：最高不超過郵政儲金二年期定期儲金年息機動利率加計年息2.175%機動計息。
    • 擔保條件：依本行「企業金融授信政策及相關規範」相關規定辦理，其資格如符合中小企業信用保證基金保證之中小企業，亦得依中小企業信用保證基金規定辦理。
