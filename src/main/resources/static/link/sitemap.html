網站導覽
無障礙操作說明

本網站的主要樣版內容分為四個大區塊：

  1. 上方主選單連結區
  2. 網頁主要內容區
  3. 左側快速連結區
  4. 下方頁尾選單連結區
  5. 如果您的瀏覽器是Firefox，快速鍵的使用方法是 Shift+Alt+(快速鍵字母)，例如 Shift+Alt+C會跳至網頁中央區塊，以此類推。

本網站的定位點 Accesskey 快速鍵設定如下：

  • Alt+U：網頁上方主選單連結區。
  • Alt+C：網頁主要內容區，此區主要顯示該網頁的主要資訊。
  • Alt+L：左側快速連結區
  • Alt+B：下方頁尾選單連結區。

本網站的Tab鍵導覽順序及頁籤以外的隱藏區塊操作：

  • 由上方的主選單連結區開始，進行至左方的主選單連結區，再移至中央的網頁主要內容區，最後移至下方的頁尾選單連結區。
  • Shift+Tab：回網頁功能之上一個項目。
  • 主選單下拉區塊預設為隱藏狀態，必須按Enter鍵方能開啟下拉區塊，開啟後，可以按ESC關閉下拉區塊。
  • 左側次選單連結區預設為隱藏狀態，必須按Enter鍵方能開啟選單，開啟後，可以按ESC關閉選單。
  • 當開啟彈出光箱後，你可以按ESC鍵關閉光箱。

本網站的左側頁籤鍵盤操作說明：

  • 當焦點進入頁籤後，請使用鍵盤的左右或上下等方向鍵來回遊走各個不同頁籤（左鍵或上鍵移往前一個，右鍵或下鍵移往後一個）
  • 當選定瀏覽頁籤後，請使用鍵盤的Tab鍵，移動已選取的頁籤焦點至其對應的內容

本網站的頁籤內容上方按鈕選單鍵盤操作說明：

  • 當焦點進入頁籤內容上方按鈕選單後，請使用Tab鍵或Shift+Tab鍵來回遊走各個不同按鈕，並使用Enter鍵選定按鈕，然後再使用Tab鍵進入選定的內容

本網站的頁籤內容上方下拉選單鍵盤操作說明：

  • 當焦點進入頁籤內容上方下拉選單後，請使用上或下方向鍵選取選單內之項目，並使用Enter鍵選定項目，然後再使用Tab鍵進入選定的項目內容，或是也可以按下ESC鍵關閉選單

本網站的頁籤內容開闔式內容區塊鍵盤操作說明：

  • 當焦點進入開闔式內容區塊後，請使用Enter鍵開啟或關閉隱藏的內容，並使用Tab鍵或Shift+Tab鍵來回遊走各個不同的開闔式內容區塊

本網站首頁輪播廣告鍵盤操作說明：

  • 當焦點進入首頁輪播廣告後，請使用Tab鍵或Shift+Tab鍵來回遊走輪播廣告的內容，前後箭頭，及底下的圓點分頁符號，使用Enter鍵可以開啟廣告連結或切換欲顯示的廣告

本網站次首頁輪播廣告鍵盤操作說明：

  • 當焦點進入次首頁輪播廣告後，請使用Tab鍵或Shift+Tab鍵來變換輪播廣告，使用Enter鍵可以開啟廣告連結

本網站首頁及次首頁頁籤操作說明：

  • 當焦點進入頁籤後，請使用Tab鍵或Shift+Tab鍵來回遊走頁籤，使用Enter鍵可將焦點移往該頁籤內容，當在頁籤內容使用Shift+Tab回歸頁籤時，請仍使用ab鍵或Shift+Tab鍵來回遊走頁籤
